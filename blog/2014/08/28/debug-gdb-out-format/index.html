2014-08-28 16:09:00

1

设置打印数组的长度上限值。如果数组中连续相同的成员的数量超过这个上限，GDB会打印字符串””，这里n是同样的重复次数，而不是重复打印这些相同的成员。将这个上限设置为0的话，打印所有的成员。默认上限时10。

显示打印重复相同成员的上限数量。

这个选项主要是设置数组的，假如你的数组太大了，那么就可以指定一个来指定数据显示的最大长度，当到达这个长度时，GDB就不再往下显示了。假如设置为0，则表示不限制。
