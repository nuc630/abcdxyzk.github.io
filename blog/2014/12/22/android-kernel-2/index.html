下面是完整版刷内核及内核模块攻略（基于恶心的Galaxy Nexus）
1. 配置交叉编译器等各种环境

1)下载交叉编译器：
$ git clone https://android.googlesource.com/platform/prebuilt

2)写入环境变量中：
export PATH=“/home/xxx/android-toolchain/prebuilt/linux-x86/toolchain/arm-eabi-4.4.3/bin:$PATH”
