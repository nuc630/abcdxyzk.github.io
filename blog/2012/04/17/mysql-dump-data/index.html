导入数据库

常用source 号令

步入mysql数据库节制台，
如mysql -u root -p
mysql>use 数据库
然后施用source号令，后面参数为脚本文件（如这里用到的.sql）

MySQL 导出表

说下上面参数
数据库用户名：root
密码 ：123456
数据库名：test
/home/my.sql 为导出文件地址，

这里 是linux系统 所以为/home/my.sql如果是windows替换 它为：e:/my.sql 注意空格奥。。
